데이터 공유 (세션에 공유)

세션을 이용해서 데이터를 공유하기 - 서블릿
	1. request객체에서 세션 id를 추출
		- 세션id를 request에서 추출해야 하므로 request객체에서 세션 id를 꺼내는 메소드를 제공한다.
		    세션id가 없는 경우 세션을 생성하는 일까지 수행한다.
		- 세션id라는 것은 서버에 만들어진 세션객체를 구분하기 위한 값
		1) getSession()
			: 새롭게 세션을 만들때 사용하는 메소드
				request 객체에서 세션 id를 추출할때 사용하는 메소드로 만약 클라이언트의 request객체에서 
				session id를 추출하지 못하는 경우 새롭게 세션객체를 서버에 만들어서 리턴한다.
											----------------------
												추후에 response할때 별도의 작업이 
												없어도 세션id를 response에 저장한다.
				session id가 존재하는 경우 기존에 사용하던 세션이 있다는 의미이므로 추출한 sesstion id
				와 일치하는 세션객체를 찾아서 세션객체를 리턴한다.
				(보통은 최초 작업인 경우 사용
				ex)로그인 하기위해 로그인 버튼을 클릭할때 세션을 새로 생성하기 위해 사용 
		2) getSession(boolean)
			- 기존 작업하는 세션이 있는지 확인하기 위해 사용
			getSession(true) - getSession()과 동일하게 동작
			getSession(false) - 기존에 사용하는 ㅔㅅ션이 있는 경우 사용하던 세션을 리턴하고 
								없으면 null을 리턴한다.
	2. 세션에 원하는 데이터를 공유
		공유하고 싶은 데이터를 세션에 추가
		setAttribute("name",value)

							java.lang.object
